# Running the app

## For API, in root folder do this:

```bash
# install dependencies
$ npm install
# development
$ npm run start
# watch mode
$ npm run start:dev
# production mode
$ npm run start:prod
```

## For UI, go to ui folder and do:

```bash
# install dependencies
$ npm install
# start vue.js ui
$ npm run serve
```
